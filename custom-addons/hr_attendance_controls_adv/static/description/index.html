This module allows the odoo Hr Attendance Manager / Administrators to define a virtual geographic boundary for attendance locations, and employees can only check in and out within one of these Geofence areas. Additionally, the module will record employee geolocation, geofence, face rRecognition with photo, IP address, and check in/out reasons.

Installation

- Copy hr_attendance_controls_adv module to addons folder

- Install the module normally like other modules

1. Check In - Check Out / Kiosk Mode configuration

Attendances -> Settings -> Check In / Check Out Control / Kiosk Mode Control


2. Geofence Configuration

Attendances -> Manager -> Geofence (Login as Admin / HR Manager) -> Create New


3. Employee Check In / Check Out

Attendances -> Check In / Check Out




3. Employee kiosk Mode

Attendances -> Check In / Check Out




4. Attendance

Attendances


5. Upload Employee Face Descriptor

Employees -> Employee - Face Recognition -> Add new Photo
